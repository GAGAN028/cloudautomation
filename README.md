# CloudAutomation — Express demo

This small Express app serves a Bootstrap 5 home page.

Quick start (using bash):

```bash
# 1) Install dependencies
npm install

# 2) Start the server
npm start

# 3) Open http://localhost:3000 in your browser
```

Files added:
- `app.js` — Express server
- `package.json` — project manifest
- `public/index.html` — Bootstrap 5 home page
